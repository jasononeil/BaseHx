class basehx.xml.XmlList
extends List<XmlNode>
A flat list of XmlNode objects. Basically just List, but with the same functions as XmlNode, which will now apply to all items in the list. Pretty neato hey!
function new(?node : XmlNode, ?list : XmlList) : Void
New Constructor that allows us to create an empty list, create one with one XmlNode element, or clone an existing list
function addList(listToAdd : List<XmlNode>) : XmlList

Append an existing XmlList to this XmlList.

Parameters: The child object to add.

Returns: this, the parent object, so we can chain methods together.

function filterByAttribute(attName : String, attValue : String) : XmlList

Return a subset of this list containing only nodes that have the matching attribute

You could possibly do regexp matches here, just check for a string that's "~/something/"

function filterByFunction(f : XmlNode -> Bool) : XmlList
Filter this list by a function. The function should take an XmlNode and return true/false.
function filterByTagName(tagName : String) : XmlList
Return a subset of this list containing only elements with this name
function getAt(index : Int) : XmlNode
Get the XmlNode at a specific index (0 based, I believe)
