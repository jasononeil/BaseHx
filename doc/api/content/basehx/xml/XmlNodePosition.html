enum basehx.xml.XmlNodePosition
import basehx.xml.XmlNode

XmlNodePosition is used to describe the relationship between two nodes. It is used by XmlNode.compareDocumentPosition()

Please note haxedoc seems to have screwed up the order of the descriptions below.

PRECEDING
The elements either don't belong to the same document or MAYBE are in unrelated parts of the DOM tree
FOLLOWING
The elements are siblings, and this element we're testing against comes after
DISCONNECTED
CONTAINS
The elements are siblings, and the element we're testing against comes first
CONTAINED_BY
The element we're testing against is a parent of this element
