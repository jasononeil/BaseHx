class AppConfig
This class contains all the config for your app. I'm not sure which package this should belong in really...
static var baseFilePath : String

Set this to the base filepath of this app on your server, (do not include a trailing slash).

Examples:

  • /var/www/html
  • /home/website
  • /home/website/myapp

This property might not be needed... not sure yet. Remember Haxe's import mechanism is alot nicer than PHPs

static var baseUrl : String

Set this to the base URL of your site, not including a trailing slash.

Examples:

  • http://www.mysite.com
  • http://www.mysite.com/myapp

static var dbDatabase : String
The name of the database to use
static var dbPassword : String
The password to use when logging in to the database server
static var dbPort : Int
The port of the database server. Usually 3306
static var dbServer : String
The address of the database server. Usually localhost
static var dbUsername : String
The username to use when logging in to the database server
static var defaultController : Class<Dynamic>
Default Controller
static var subDir : String
The subdirectory on the server we are in. No trailing slash
static var tablePrefix : String
If you have multiple apps (or versions of apps) on this database, you can specify a table prefix to keep table names from overlapping each other. For example, "myapp1.0_"
